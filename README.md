# Como rodar o projeto
## 1 - Preparar o ambiente de desenvolvimento (CLI)
Instruções na [página official do react native](https://reactnative.dev/docs/set-up-your-environment).
## 2 - Instalar as dependencias
Na pasta do projeto:
> npm install -s
## 3 - Rodar o projeto
Iniciar o servidor:
> npm start

Iniciar a aplicação:
> npm run android
